Angular-Widgetbox

A module for angularJS that lets you place widgets into boxes and rearrange them by dragging them and dropping them in other boxes. No dependencies except for angularJS and document.querySelectorAll. If you've ever used iGoogle or My Yahoo - you know what I'm talking about.

Documentation and source code on Github

Quick Install bower install angular-widgetbox

Microload

A very tiny (less than 1KB) async script/stylesheet loader. You can load scripts consecutively and in parallel, then optionally execute a callback once all is done.

Documentation and source code on Github

Quick Install bower install microload
