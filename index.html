  • Angular-Widgetbox

    A module for angularJS that lets you place widgets into boxes and rearrange them by draggin them and dropping them in other boxes. No dependencies except for angularJS and document.querySelectorAll. If you've ever used iGoogle or my.yahoo - you know what I'm talking about.

    Checkout the Demo

    Documentation and source code on Github

    Quick Install bower install angular-widgetbox
